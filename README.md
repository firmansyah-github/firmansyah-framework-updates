Welcome, Software Architects and Developers! **The Factor** is the ultimate enterprise-grade database-to-API code generation engine for Eclipse IDE. By bridging your normalized database schema directly with deterministic, template-driven generation, The Factor eliminates thousands of lines of monotonous CRUD boilerplate in seconds, offering a scale, speed, and consistency that probabilistic AI tools simply cannot match.

## 📖 Project Overview & Documentation Summary

For an exhaustive setup guide, architectural specs, and blueprint customizer manuals, visit the:
👉 **[Single Comprehensive Doc Portal](https://firmansyah-github.github.io/factor.docs/#/)**

### Key Capabilities:
* **JDBC Introspection**: Automatically reads database metadata (data types, primary keys, composite keys, unique indexes, self-referencing constraints, and complex foreign key relationships) directly from PostgreSQL, MySQL, Oracle, SQL Server, and more.
* **Apache FreeMarker Blueprinting**: Yields 100% compliant, standard-oriented, and perfectly consistent Java codebases (JPA Entities, Repositories, DTOs, Mappers, Services, Controllers, and Integration Tests) strictly governed by custom `.ftl` templates.
* **"Tool Form" Code Reverse Engineering**: Paste your hand-crafted REST Controller or Service template into the **Tool Form**, and the plugin automatically reverse-engineers the code structure into a reusable FreeMarker blueprint!
* **Enterprise Velocity**: Proven to generate over **50,000+ lines of pristine, commented, and compilable Java code** for a moderate schema of 80 tables in less than 5 minutes—saving 120+ developer hours per project.
